偽代碼 (Pseudocode)

FUNCTION startAnimation(目標值):
  停止任何正在運行的動畫

  取得 目前顯示的數字
  取得 平滑係數 (speed)
  設定 更新間隔 (interval = 20ms)
  設定 上次更新時間 = null

  // 定義動畫的每一步要做什麼
  FUNCTION animationStep(當前時間):
    如果 上次更新時間 是 null, 則設定 上次更新時間 = 當前時間

    計算 經過時間 = 當前時間 - 上次更新時間
    計算 剩餘距離 = 目標值 - 目前顯示的數字

    如果 剩餘距離的絕對值 < 0.5 (已到達終點):
      設定 顯示數字 = 目標值
      結束動畫並返回
    
    如果 經過時間 >= 更新間隔:
      計算 移動量 = 剩餘距離 * 平滑係數

      如果 移動量的絕對值 < 1 (避免停滯):
        移動量 = 剩餘距離的符號 (1 或 -1)

      更新 目前顯示的數字 += 移動量
      更新 畫面上的數字為四捨五入後的目前值

      重設 上次更新時間 = 當前時間

    請求下一幀動畫，繼續執行 animationStep

  // 啟動動畫
  啟動 animationStep
END FUNCTION

let animationFrameId: number | null = null;

function startDynamicAnimation(targetValue: number): void {
    if (animationFrameId) {
        cancelAnimationFrame(animationFrameId);
    }

    let currentValue: number = /* 從 DOM 取得目前數字 */;
    const speed: number = /* 從滑桿取得平滑係數 */;

    let lastUpdateTime: number | null = null;
    const interval: number = 20; // 更新間隔 (ms)

    const step = (currentTime: number): void => {
        if (lastUpdateTime === null) {
            lastUpdateTime = currentTime;
        }

        const elapsed = currentTime - lastUpdateTime;
        const distance = targetValue - currentValue;

        if (Math.abs(distance) < 0.5) {
            /* 更新 DOM 為最終目標值 */;
            animationFrameId = null;
            return;
        }

        if (elapsed >= interval) {
            let movement = distance * speed;

            if (Math.abs(movement) < 1) {
                movement = Math.sign(distance);
            }
            
            currentValue += movement;
            /* 更新 DOM 為 Math.round(currentValue) */;
            
            lastUpdateTime = currentTime;
        }
        
        animationFrameId = requestAnimationFrame(step);
    };

    animationFrameId = requestAnimationFrame(step);
}
